About

GenOS is a Linux distribution.

It's goal is to be easy to use and fast.

It is easy to switch from another operating system to GenOS thanks to it's intuitive interface and compatibility.

Open source

We believe every user should be free to download, change and share their software for any purpose.

Because of that, GenOS is built with open source software.
